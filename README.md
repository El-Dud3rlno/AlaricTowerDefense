# AlaricTowerDefense

A tower defense game featuring boss-only rifts with various tower types and strategic gameplay.

## Live Demo

The game is deployed via GitHub Pages and will be available at:
**https://el-dud3rlno.github.io/AlaricTowerDefense/**

## How to Play

1. Visit the live demo URL above
2. Click and drag to place towers on the map
3. Use the tower selection button to choose different tower types (Fast, Heavy, Dual, Fire, Ice, Lightning, Poison)
4. Click "Start Rift" to begin a wave
5. Defend against increasingly difficult boss waves

## Tower Types

- **Fast** (25 points): Quick firing, low damage
- **Heavy** (50 points): Slow firing, high damage with splash
- **Dual** (100 points): Fires two projectiles
- **Fire** (40 points): Burning damage over time
- **Ice** (35 points): Slows enemies
- **Lightning** (45 points): Chain lightning effect
- **Poison** (40 points): Poison damage over time

## Deployment

The game is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch. The workflow configuration can be found in `.github/workflows/deploy.yml`.

## Local Development

Simply open `index.html` in a web browser to play locally.
