# Alaric Tower Defense

A Diablo Rift-style loot-first tower defense prototype.

## Project layout
- `index.html` / `Index.html`: playable build
- `src/config/balance.js`: centralized balance config
- `docs/ARCHITECTURE.md`: structure notes
- `docs/BALANCE.md`: tuning workflow

## Running
Open `index.html` in a browser.

## Balancing
Tune values in `src/config/balance.js` and reload.
